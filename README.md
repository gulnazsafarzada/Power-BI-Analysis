📊 Power BI Analizləri
Bu repozitoriyada müxtəlif sahələr üzrə məlumat analizi və vizuallaşdırmaları əhatə edən Power BI hesabatları toplanmışdır. Hər bir hesabat məlumatların təmizlənməsindən başlayaraq interaktiv hesabatların yaradılmasına qədər olan tam prosesi əhatə edir.


🛠 İstifadə olunan texnologiyalar:

Power Query: Məlumatların ETL (təmizlənməsi və transformasiyası) prosesi üçün.

DAX (Data Analysis Expressions): Mürəkkəb hesablamalar və dinamik ölçülərin (measures) yaradılması üçün.

Data Modeling: Cədvəllər arası əlaqələrin (Star Schema/Snowflake Schema) qurulması.

Vizuallaşdırma: İstifadəçi dostu və məna kəsb edən dinamik vizualların dizaynı.
